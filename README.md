# Shape Editor

A modern, interactive shape editor application built with React and Vite. Create, customize, and manipulate various shapes with an intuitive interface.

## Features

### Shape Tools
- **Rectangle** - Create and edit rectangular shapes
- **Square** - Draw perfect square shapes
- **Circle** - Create circular shapes
- **Triangle** - Add triangular elements to your canvas

### Customization Options
- **Fill Color** - Choose from a palette of preset colors or enter custom hex values
  - Preset colors: Blue, Purple, Pink, Red, Orange, Yellow, Green, Teal, Light Blue, Dark Blue, and White
  - Custom color support via hex input
- **Stroke Controls** - Adjust stroke width with an interactive slider
- **Canvas Interactions**:
  - Click to deselect shapes
  - Drag to move and reposition shapes
  - Scroll to zoom in/out
  - Alt+Drag to pan around the canvas
  - Delete/Backspace to remove selected shapes

### Editor Controls
- Zoom controls (zoom in/out buttons and percentage display)
- Reset view to 100%
- Real-time shape counter

## Getting Started

1. Open `index.html` in your browser
2. Select a shape from the SHAPES panel on the left
3. Click on the canvas to place the shape
4. Customize the appearance using:
   - Fill color palette or custom hex color
   - Stroke width slider
5. Interact with shapes:
   - Drag to move
   - Use keyboard shortcuts for advanced actions
